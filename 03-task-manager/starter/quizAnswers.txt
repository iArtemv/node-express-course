Quickly explain how middleware is used in an Express application.
Middleware functions are used in Express to handle requests, process data, and provide responses. 
They can perform tasks such as logging, authentication, parsing request bodies, and handling errors. 
Middleware functions are executed sequentially for each request.

What does CRUD stand for?
CRUD stands for Create, Read, Update, and Delete. 
These are the four basic operations that can be performed on data in a database.
